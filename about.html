About Pawmart

Pawmart is a demo web application designed to showcase user authentication, simple admin tools, and a product browsing experience for pet lovers.

What is Pawmart?

Pawmart is built as a course project for COMP229 – Web Application Development. The application simulates an online hub where pet owners can sign up, log in, view curated products, and manage their basic profile.

Key Features

  • Secure user registration and login with JWT-based authentication.
  • Role-based access: admin users can view and manage registered accounts.
  • Profile page showing user information stored in the system.
  • Products page with curated items for pets, organized for easy browsing.

Technology Stack

Pawmart uses a Node.js and Express backend with MongoDB for data storage. The frontend is built with HTML, CSS, and vanilla JavaScript, focusing on clean structure and clear separation between views and API calls.

Project Purpose

The main goal of this project is to practice building a full-stack web application using an MVC structure, RESTful APIs, authentication and authorization, and a basic external design suitable for future enhancements.
